Setup Instructions

  1. Open the WebSophon extension popup on this page
  2. Enter a webhook URL (can be a test URL like https://webhook.site/xxx)
  3. Add at least one evaluation field with criteria
  4. Toggle "Enable screenshot capture" to start automatic captures
  5. Or click "Capture Screenshot Now" for manual captures
  6. Open the extension popup again and check the Event History
